      Parameters

      • battlerIndex: BattlerIndex

        The BattlerIndex of the Pokemon obtaining the status effect.

      • statusEffect: StatusEffect

        The StatusEffect being applied.

      • sourcePokemon: null | Pokemon = null

        The Pokemon applying the status effect to the target, or null if the status is applied from a non-Pokemon source (hazards, etc.); default null.

      • OptionalsleepTurnsRemaining: number

        The number of turns to set StatusEffect.SLEEP for; defaults to a random number between 2 and 4 and is unused for non-Sleep statuses.

      • sourceText: null | string = null

        The text to show for the source of the status effect, if any; default null.

      • statusMessage: string = ""

        A string containing text to be displayed upon status setting; defaults to normal key for status if empty or omitted.

      Returns ObtainStatusEffectPhase
