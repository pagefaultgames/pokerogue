    pendingStatus

    pendingStatus: StatusEffect = StatusEffect.NONE

    Tracker for a pending status effect

    Remarks

    Set whenever Pokemon#trySetStatus succeeds in order to prevent subsequent status effects from being applied. Necessary because the status is not actually set until the ObtainStatusEffectPhase runs, which may not happen before another status effect is attempted to be applied.
