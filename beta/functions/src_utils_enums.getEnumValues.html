    Function getEnumValues

    • getEnumValues<E extends EnumOrObject>(enumType: TSNumericEnum<E>): E[keyof E][]

      Return the numeric values of a numeric Enum object, excluding reverse-mapped strings.

      Type Parameters

      • E extends EnumOrObject

      Parameters

      • enumType: TSNumericEnum<E>

        The enum object to retrieve keys for

      Returns E[keyof E][]

      An ordered array of all of enumType's number values

      Example

      enum fruit {
        apple = 1,
        banana = 2,
        cherry = 3,
        orange = 12,
      };

      console.log(getEnumValues<typeof fruit>(fruit)); // output: [1, 2, 3, 12]

      Remarks

      To retrieve the keys of a NormalEnum, use ObjectConstructor.values instead.
