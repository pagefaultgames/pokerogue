    Type Alias MatterRunnerConfig

    Configuration for the Matter Physics Runner.

    Set only one of fps and delta.

    delta is the size of the Runner's fixed time step (one physics update). The "frame delta" is the time elapsed since the last game step. Depending on the size of the frame delta, the Runner makes zero or more updates per game step.
