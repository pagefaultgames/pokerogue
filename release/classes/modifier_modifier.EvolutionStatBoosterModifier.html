    Class EvolutionStatBoosterModifier

    Modifier used for held items, specifically Eviolite, that apply Stat boost(s) using a multiplier if the holder can evolve.

    apply

    • apply(pokemon: Pokemon, stat: Stat, statValue: NumberHolder): boolean

      Boosts the incoming stat value by a EvolutionStatBoosterModifier.multiplier if the holder can evolve. Note that, if the holder is a fusion, they will receive only half of the boost if either of the fused members are fully evolved. However, if they are both unevolved, the full boost will apply.

      Parameters

      • pokemon: Pokemon

        Pokemon that holds the item

      • stat: Stat
      • statValue: NumberHolder

        NumberHolder that holds the resulting value of the stat

      Returns boolean

      true if the stat boost applies successfully, false otherwise
